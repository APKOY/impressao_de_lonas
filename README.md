# LonaBoard 🖨️

Um aplicativo web estilo Kanban, leve e em tempo real, desenvolvido para gerenciar o fluxo de produção de gráficas e comunicação visual (lonas, banners, adesivos, etc.).

## 🚀 Funcionalidades

* **Quadro Kanban Visual:** Etapas claras de produção (A Fazer, Em Andamento, Acabamento, Concluído).
* **Gestão de Pedidos:** Cards com prioridade, responsável, tags de produto e upload de prints/provas de arte com compressão automática.
* **Equipe e Alertas:** Cadastro de membros com avatares coloridos e histórico de notificações de movimentação de cards.
* **Tempo Real:** Sincronização instantânea entre todos os usuários utilizando Firebase.
* **Responsivo:** Layout mobile-first com *Dark Mode* nativo.

## 🛠️ Tecnologias

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Backend/BaaS:** Firebase (Firestore)
